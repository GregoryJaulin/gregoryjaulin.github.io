Fuze'Bot

Bot development for a french Youtuber's Discord server.

 What is Discord ?
Discord is an American proprietary freeware instant messaging and VoIP application and digital distribution platform designed for creating communities ranging from gamers.

Mostly created to help server's moderation, Fuze'bot react to some types of messages in order to automate different kind of actions
It is also able to act alone if necessary, for example to punish spam, delete messages, give roles or even back up some kind of actions performed on the server.

Developed in Java, I maintained it with Debian 9 puis 10. MongoDB has been choosen to administrate all databases because of his ease to manage complexes data with object encapsulation.

Minecraft Plugins

Mini-games oriented plugin development for a Minecraft server

 What is a Plugin ?
A plugin is an additive file for a Minecraft multiplayer server. Plugins are often used to change certain aspects of the game.

Mainly for recreational purposes, this plugin was planning several mini-games :
• Hide & seek
 • Werewolf
 • Battle Royale
 • ...

 This plugin was using version 1.8.8 of Spigot, based on an event system. MongoDB has been used to manage all different databases required to back up games stats and players inventory.
